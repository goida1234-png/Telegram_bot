## ✨ Функционал

### 🏠 Главное меню

Стартовый экран после команды `/start`. Бот автоматически регистрирует пользователя по его Telegram ID — никакой дополнительной авторизации не требуется. Доступны четыре раздела: начать викторину, посмотреть статистику, экспорт/импорт данных и информация о боте.

### 📚 Выбор главы

Четыре тематических раздела на выбор. Каждая глава содержит **20 вопросов**, которые перемешиваются случайно при каждом прохождении — можно проходить повторно и каждый раз будет по-новому.

| Глава           | Вопросов |
| --------------- | -------- |
| 🎮 Dota 2       | 20       |
| 🔫 CS2 / CS:GO  | 20       |
| ⚽ Футбол       | 20       |
| 🧠 Общие знания | 20       |

### ❓ Викторина

- Каждый вопрос имеет **4 варианта ответа** (A, B, C, D)
- Прогресс-бар показывает сколько вопросов пройдено: `[🟩🟩🟩⬜⬜⬜⬜⬜⬜⬜] 3/20`
- После каждого ответа бот показывает реакцию — ✅ Правильно или ❌ Неверно с указанием правильного варианта
- Текущий счёт виден на каждом шаге

### 🏁 Экран результата

По окончании викторины отображается:

- Количество правильных и неверных ответов
- Процент правильных ответов
- Звёздный рейтинг (⭐ из 5)
- Оценка результата (от 😅 до 🏆)
- **Разбор ошибок** — до 5 вопросов с твоим и правильным ответом

### 📊 Статистика

Личная статистика по каждой пройденной главе:

- Количество сыгранных игр
- Лучший результат
- Средний процент правильных ответов
- Звёздный рейтинг

### 🚪 Выход из викторины

Кнопка выхода доступна на каждом вопросе. Перед выходом бот показывает **диалог подтверждения** — можно передумать и продолжить с того же места.

### 💾 Сохранение данных

Каждый ответ сохраняется в базу данных SQLite в реальном времени:

- Пользователь и его Telegram ID
- Сессия прохождения с временем начала и конца
- Каждый ответ: вопрос, выбранный вариант, правильный вариант, результат

### 📤 Экспорт результатов в XML

Все результаты можно выгрузить в XML файл одним нажатием кнопки. Файл содержит полную историю прохождений:

```xml
<?xml version='1.0' encoding='utf-8'?>
<quiz_results exported_at="2024-01-15 14:30:00" version="1.0">
  <user>
    <tg_id>123456789</tg_id>
    <username>username</username>
    <first_name>Имя</first_name>
  </user>
  <sessions total="2">
    <session chapter="dota2" score="18" total="20" percent="90%">
      <answers>
        <answer index="0" is_correct="1">
          <question>Какой герой имеет способность «Ravage»?</question>
          <chosen>A. Tidehunter</chosen>
          <correct>A. Tidehunter</correct>
        </answer>
        ...
      </answers>
    </session>
  </sessions>
</quiz_results>
```

Имя файла формируется автоматически: `quiz_results_{tg_id}_{дата}.xml`

### 📥 Импорт результатов из XML

Загрузи ранее сохранённый XML файл обратно в бота — все сессии восстановятся в базе данных и отобразятся в статистике. Удобно при переносе данных или смене устройства. Данные добавляются к существующим, не перезаписывая их.

```
┌──────────────┐       ┌─────────────────┐       ┌──────────────────┐
│    users     │       │  quiz_sessions  │       │     answers      │
├──────────────┤       ├─────────────────┤       ├──────────────────┤
│ id           │──┐    │ id              │──┐    │ id               │
│ tg_id        │  └───▶│ user_id         │  └───▶│ session_id       │
│ username     │       │ chapter         │       │ question_index   │
│ first_name   │       │ score           │       │ question_text    │
│ joined_at    │       │ total           │       │ chosen_option    │
└──────────────┘       │ started_at      │       │ correct_option   │
                       │ finished_at     │       │ is_correct       │
                       └─────────────────┘       └──────────────────┘
```

## ⚙️ Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/username/quiz_bot.git
cd quiz_bot
```

### 2. Создай виртуальное окружение

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate
```

### 3. Установи зависимости

```bash
pip install aiogram==3.7.0 aiosqlite python-dotenv
```

### 4. Создай `.env` файл

```bash
cp .env.example .env
```

Вставь свой токен от [@BotFather](https://t.me/BotFather):

```env
BOT_TOKEN=your_token_here
```

### 5. Запусти бота

```bash
python main.py
```
